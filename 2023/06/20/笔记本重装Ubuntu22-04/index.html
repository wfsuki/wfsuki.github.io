笔记本重装Ubuntu22.04

发表于2023-06-20|更新于2023-06-20
|阅读量:

给闲置的旧笔记本安装一个Ubuntu，当做个人服务器，进行一波废物利用😉

制作启动U盘

  1. 先看Ubuntu官方教程，总结就是使用Rufus制作一个启动盘，当然也可以用别的气动工具

    Create a bootable USB stick with Rufus on Windows | Ubuntu

  2. 下载镜像在这里

    Download Ubuntu Desktop | Download | Ubuntu

进入系统BIOS界面

接下来就是重启电脑，一直按Delete或者F9之类的，和一般安装windows一样，从U盘启动并安装系统即可
